# Verwalten von Benachrichtigungen in Microsoft Intune
Verwenden Sie den Arbeitsbereich **Benachrichtigungen** der Administrator Intune-Verwaltungskonsole aus, um den allgemeinen Zustand des Geräte in Ihrer Organisation bewerten und Probleme erkennen.

## Aktive Benachrichtigungen anzeigen

Finden Sie unter Allgemeine Informationen und Zusammenfassung Daten zu aktiven Benachrichtigungen.

#### Aktive Benachrichtigungen anzeigen

Führen Sie eine der folgenden Folge von Schritten in der Intune Admin-Verwaltungskonsole:

-  Wählen Sie **Allgemeine Informationen zu Benachrichtigungen angezeigt**, einschließlich der oberen drei Benachrichtigungen und die Gesamtzahl der aktiven Benachrichtigungen **Systemüberblick**aus. Wählen Sie in dieser Benachrichtigungen Ausführlichere Informationen finden Sie unter den Links aus.

-  **Zusammenfassungsdaten für Benachrichtigungen angezeigt werden**, wählen Sie **Benachrichtigungen** > **Übersicht**. Klicken Sie auf der Seite **Übersicht Benachrichtigungen** können Sie eine Zusammenfassung der Benachrichtigungen im Bereich **Benachrichtigen Typ Zusammenfassung** anzeigen. Kritische Benachrichtigungen werden zuerst aufgeführt. Wählen Sie in dieser Benachrichtigungen Ausführlichere Informationen finden Sie unter den Links aus.

        > [!NOTE]
        > In some cases, an alert type might appear more than once in the **Alerts Type Summary** list.
        >
        > For example, two instances of the Logical Free Disk Space alert type might be listed:
        >
        > -   3 Logical Free Disk Space
        > -   2 Logical Free Disk Space
        >
        > This occurs when the same alert type is generated for different devices that run different operating systems. In the example, the 3 Logical Free Disk Space alert type might have been generated by a computer that runs Windows 7. The 2 Logical Free Disk Space alert type might have been generated by a computer that runs Windows Vista.

-   **Alle aktive Benachrichtigungen angezeigt werden**, wählen Sie **Benachrichtigungen** > **Alle Benachrichtigungen**. Klicken Sie auf der Seite **Benachrichtigungen** können Sie eine Liste aller aktiven Benachrichtigungen finden Sie unter. Die Seite weist die folgenden Spalten:

    -   **Namen**. Dies ist der Name des Typs benachrichtigen, der die Warnung generiert.

    -   **Quelle**. Diese Spalte enthält einen Link zur Quelle der Warnung an. Wenn Sie der oberen Schwellenwert der Anzeige des Benachrichtigungssounds Typs auf **Alle anzeigen**festgelegt ist, zeigt diesen Link ein einzelnes Gerät aus. Wenn der oberen Schwellenwert der Anzeige des Benachrichtigungssounds Typs auf einen Wert festgelegt ist, zeigt den Link eine Liste aller Geräte, die durch diese Warnung beeinträchtigt wird.

    -   **Letzte Aktualisierung**. Dies weist darauf hin, die Uhrzeit der letzten die Benachrichtigung Änderung. Wenn eine Warnung geschlossen ist, wird die Uhrzeit die Benachrichtigung geschlossen wurde angezeigt.

    -   **Schwere**. Diese Spalte gibt die Schwere der Warnung an.

## Ansicht Schwarzen Brett Benachrichtigungen
Schwarzen Brett Benachrichtigungen bieten wichtige dienstankündigungen. Sie können Informationen zu einer anstehenden dienstupgrade oder Wartung Terminplan oder den Status von einem Ausfall bieten.

#### Zum Anzeigen und schwarzen Brett Benachrichtigungen verwalten

1.  Wählen Sie in der Verwaltungskonsole Intune **Systemüberblick**aus.

2.  Falls dienstankündigungen wichtig sind, werden ihnen im Bereich **Schwarzen Brett** angezeigt.

3.  Wenn Sie eine Benachrichtigung Schwarzen Brett zu einer durch Trennzeichen getrennte Werte (CSV) oder HTML-Datei, in der der Verwaltungskonsole Intune exportieren möchten, wählen Sie **Benachrichtigungen** > **Alle Benachrichtigungen** >    **bemerkt**. Wählen Sie eine Mitteilung, wählen Sie das Symbol für die **Liste exportieren** aus, und folgen Sie dann die Anweisungen angezeigt.

## Überprüfen der Systemstatus Intune
Klicken Sie im Arbeitsbereich **Systemüberblick** können Sie **Systemstatus** Übersichten für die Kategorien Endpunkt Schutz, Updates, Agent Gesundheit, Richtlinie und Software, damit Sie erkennen und Priorisieren Sie Probleme, die sofortige Aufmerksamkeit anzeigen. Fehlermeldungen, die generiert werden, wenn das System unterbrochen Link zur Zusammenfassung **Dienststatus** . **Dienststatus** Zusammenfassung zeigt Details zu dem Problem an jeder Position und die Uhrzeit der letzten der Status Zusammenfassung wurde aktualisiert.

#### Anzeigen des Status Ihres Abonnements

1.  Wählen Sie in der Verwaltungskonsole Intune **Systemüberblick**aus.

2.  Untersuchen Sie den **Systemstatus** Bereich für den Status der verschiedenen Komponenten Microsoft Intune.

  Viele Elemente sind verknüpft, sodass Sie weitere Informationen sehen können. Angenommen, klicken Sie unter **Endpunkt Schutz**sehen, wenn Sie die Anzahl der Instanzen, wählen Sie des Arbeitsbereichs **Endpunkt-Schutz** mit einer Liste von Schadsoftware, die erkannt wurde. Wenn Sie die Anzahl der Geräte auswählen, können Sie sehen, klicken Sie im Arbeitsbereich **Gruppen** mit einer Liste von Geräten auf denen Schadsoftware gefunden wurde.

## Schließen und reaktivieren Benachrichtigungen
Intune Benachrichtigungen bleiben aktiv, bis Sie eines der folgenden Ereignisse eintritt:

-   Das Problem, das die Benachrichtigung generiert werden hat eine feste Position.

-   Die Benachrichtigung wird manuell geschlossen.

-   Fünf Tage übergeben, nachdem die Warnung erzeugt wurde. Die Benachrichtigung nach 45 Tagen läuft ab, und wird automatisch geschlossen.

Benachrichtigungen, die als abgeschlossen markiert sind, werden nach 90 Tagen endgültig gelöscht.

#### Um eine Warnung manuell zu schließen.

Führen Sie eine der folgenden Folge von Schritten in der Intune Admin-Verwaltungskonsole:

- **Um eine Warnung aus der Liste der Benachrichtigungen zu schließen**, wählen Sie **Benachrichtigungen** > **Alle Benachrichtigungen**. Wählen Sie eine Warnung, und wählen Sie dann auf **Schließen Benachrichtigung**.

- **Um eine Benachrichtigung für ein bestimmtes Gerät zu schließen**, wählen Sie die **Gruppen** > **Alle Geräte**. Wählen Sie ein Gerät aus, und wählen Sie dann auf **Eigenschaften anzeigen**. Klicken Sie dann auf die Registerkarte **Benachrichtigungen** , wählen Sie eine Warnung, und wählen Sie dann auf **Schließen Benachrichtigung**.

- **Schließen einer Schwarzen Brett** **Systemüberblick**auswählen. Wählen Sie das **X** neben der Benachrichtigung Schwarzen Brett aus.

#### Zum Anzeigen und geschlossene Benachrichtigungen reaktivieren

1.  Wählen Sie in der Verwaltungskonsole Intune **Benachrichtigungen** > **Alle Benachrichtigungen**.

2.  Wählen Sie in der Liste **Filter** **geschlossen**.

    Sie können die Namen und Weitere Informationen zu den Benachrichtigungen im Listenbereich Management finden Sie unter. Anzeigen von Details zur ausgewählten Warnung im Vorschaufenster.

3.  Um die ausgewählte Warnung reaktivieren möchten, wählen Sie **Reaktivieren Warnung**aus.
